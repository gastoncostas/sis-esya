# Aula Virtual - Escuela de Suboficiales y Agentes

## Instrucciones de Uso

1. Requisitos:
   - PHP 7.4+ (recomendado 8.0+)
   - Permisos de escritura en /data y /uploads

2. Usuarios demo:
   - Admin: admin / Admin1234
   - Profesor: profesor / Prof1234
   - Alumno: alumno / Alumno1234

3. Estructura de módulos:
   - Módulo 1: Formación básica (siempre activo)
   - Módulo 2: Especialización (activar desde Panel Admin)
   - Módulo 3: Prácticas (en desarrollo)

## Para desarrolladores
- Los archivos se guardan en /uploads/<módulo>/<materia>
- La configuración en /data/settings.json
- Usuarios en /data/users.json
